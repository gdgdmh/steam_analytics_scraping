## 使い方
### pipenvの起動
* `pipenv shell`
### pipenvの終了
* `deactivate`
* `exit`
### 実行(ディレクトリに移動して)
`python -m sas`
### テスト実行(ディレクトリに移動して)
`pytest`
### 特定のテストのみ実行
`pytest tests/sample.py`
### Help
`python -m sas`や`pytest`などのコメントが効かない時は  
`pip install requests`  
`pip install bs4`  
`pip install pytest`
